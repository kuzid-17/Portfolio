# Normalized Cyclic Convolution over GF(2^m)

## Описание

Данный проект реализует **нормализованную циклическую свёртку** над конечным полем `GF(2^m)` для чётных `m = 4, 6, 8`.

Алгоритм основан на методе, предложенном в статье:

> **S. V. Fedorenko**, *"Normalized Cyclic Convolution: The Case of Even Length"*,  
> IEEE Transactions on Signal Processing, vol. 63, no. 20, pp. 5308–5315, Oct. 2015.

Проект **индивидуальный**, все модули написаны автором.

---

## Реализованный функционал

- **Поле GF(2^m)** — сложение, умножение, возведение в степень
- **Подсчёт операций** — отдельный счётчик для умножений и сложений в поле
- **Наивная свёртка** — умножение вектора на полную циркулянтную матрицу
- **Быстрая свёртка** — последовательное умножение на разреженные матрицы
- **Верификация** — сравнение результатов наивного и быстрого методов

---

## Зависимости

Проект использует только **стандартную библиотеку Python**:

- `random` — генерация случайных векторов
