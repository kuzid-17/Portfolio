# Portfolio

Репозиторий представляет собой коллекцию небольших проектов и примеров кода, написанных на разных языках программирования.

## 🚀 О репозитории

В работе над проектами использовались различные библиотеки, форматы данных и математические методы, а кодовая база демонстрирует навыки моделирования систем передачи, работы с конечными полями, реализации криптографических алгоритмов и создания микросервисных приложений.

Репозиторий служит техническим портфолио для демонстрации подхода к разработке, структурированию кода и решению прикладных задач.

---

## 📂 Структура

Каждая папка содержит отдельный проект со своим файлом README.

| Проект | Описание |
| :--- | :--- |
| **Блочные коды** | Моделирование работы блочных кодов в PUCCH FORMAT2 |
| **Свёрточные коды** | Программная реализация различных алгоритмов декодирования свёрточного кода |
| **Генератор псевдослучайных чисел** | Криптостойкий генератор случайных последовательностей и алгоритм шифрования данных |
| **Микросервис Коннектор** | Один из основных компонентов для системы анализа задач из репозиториев Jira |
| **Циклическая свёртка** | Сравнение методов вычисления циклической свёртки над конечными полями |
