# BBS Project

## Описание

Проект реализует:

- Криптостойкий генератор псевдослучайных чисел **Blum–Blum–Shub (BBS)**  
- Тестирование на псевдослучайность по стандарту **FIPS 140-1**  
- Шифрование **Blum–Goldwasser**

---

## Зависимости

Установить GMP (Linux):

```bash
sudo apt-get install libgmp-dev
```

---

## Сборка

Проект собирается через Makefile:

```bash
make
```

После сборки создаётся исполняемый файл: `bbs_project`

---

## Запуск

```bash
./bbs_project
```

---

## Формат вывода

Программа последовательно выполняет 3 сценария:

1.  **BBS + FIPS 140-1 тесты**
    *   Генерация последовательности из 20 000 бит
    *   Прохождение тестов: *Monobit*, *Poker*, *Runs*, *Long Run*
2.  **Проверка примера из Handbook of Applied Cryptography (Example 8.57)**
    *   Сравнение полученного шифра с эталонным
    *   Проверка корректности дешифрования
3.  **Демонстрация Blum–Goldwasser**
    *   Генерация ключей
    *   Шифрование произвольной строки
    *   Восстановление исходного сообщения
