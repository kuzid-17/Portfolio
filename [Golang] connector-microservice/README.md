# JIRA Analytics Connector (Microservice)

## Описание

Данный проект является частью групповой разработки микросервисного приложения для анализа данных из JIRA.

⚠️ В репозитории представлен **только микросервис Коннектор**, реализованный автором, для ознакомления. Остальные компоненты системы не включены. Полноценное приложение в рамках данного репозитория не собирается.

## Назначение сервиса

Коннектор предназначен для получения данных из JIRA через REST API, их нормализации и подготовки, а также выгрузки в локальную базу данных.

## Особенности

- независимый микросервис с модульной структурой
- потоковая обработка данных
- демонстрирует работу с внешними API, построение микросервисов и подготовку данных для аналитических систем
