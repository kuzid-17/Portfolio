# Convolutional Codes

## Описание

Данный проект является частью групповой разработки, посвящённой моделированию работы свёрточных кодов.

⚠️ В репозитории представлен **только код той части, которую реализовал автор**, исключительно для ознакомления и демонстрации навыков.  
Реализации других участников команды намеренно не включены. Полноценная сборка и запуск не предполагаются.

---

## Реализованный функционал

### Модуляция и демодуляция

Поддержаны цифровые схемы модуляции:

- BPSK  
- QPSK  
- QAM-16, QAM-64, QAM-256  

Реализованы как **жёсткие**, так и **мягкие** решения на выходе демодулятора, позволяющие использовать вероятностную информацию при декодировании.

---

### Кодирование и декодирование

Реализованы различные варианты:
- свёрточные коды на основе **усечённой** решётки
- классический алгоритм Витерби
- поддержка **tail-biting** кодирования
- алгоритмы декодирования для tail-biting кодов
- списочное декодирование
